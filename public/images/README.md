# Images for Partner View

Please add the following images to this directory:

1. **bomb-squad.jpg** - Image of a bomb disposal technician in full protective gear, kneeling and carefully handling something (for Nurture Phase - super cautious days, shown with "PROCEED WITH EXTREME CAUTION" message)
2. **play-dead-animal.jpg** - Image of a prairie dog lying on its back playing dead on reddish-brown dirt (for the 2-3 most critical days before period, shown with "PLAY DEAD" message)

The images should be:
- JPG or PNG format
- Recommended size: 600x400px or similar aspect ratio
- The component will automatically resize them
- Save them with the exact filenames: `bomb-squad.jpg` and `play-dead-animal.jpg`

If images are not provided, the component will gracefully hide the image container.
